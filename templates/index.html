About This Project

This system uses machine learning and metadata analysis to recommend the most suitable datasets based on your query, preferences, and project domain.

Key Features

🔍 Smart Matching

Find datasets that match your needs using KNN and metadata.

⚙️ Lightweight Model

Fast recommendations based on preprocessed and scaled features.

📁 Open Data Sources

Works with datasets from Kaggle, UCI, and public repositories.

How It Works

  1. Enter a dataset name you're familiar with or just a topic.
  2. The system finds the most similar datasets using KNN + TF-IDF.
  3. Results are displayed instantly in a popup.
  4. Click and explore your recommended datasets!
